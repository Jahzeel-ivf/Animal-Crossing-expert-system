# 🦉 Animal Crossing Expert System

An expert system that recommends villagers for your island in **Animal Crossing: New Horizons** based on your personal preferences.

This project was developed as part of a **school project** and for **entertainment purposes**, exploring the design of rule-based systems and decision-making logic.

---

## 🌿 About the Project

Choosing villagers for your island can be difficult because each one has different:

* personalities
* species
* styles
* colors
* interactions

This expert system helps players **find villagers that match their preferences** by analyzing these characteristics and applying **rule-based logic**.

The system asks the user a series of questions and then recommends villagers that best fit the selected preferences.

---

## 🧠 How It Works

The expert system evaluates different attributes of each villager:

* Personality
* Species
* Decorative style
* Color
* Compatibility with other villagers

Based on the answers provided by the user, the system applies **decision rules** to generate personalized recommendations.

---

## 📊 Data Collection

The villager data used in this project was obtained from:

https://acnh.co/

To collect this information, a **web scraping program written in Python** was developed to extract relevant villager data automatically.
